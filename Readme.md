🤖 GenAI Financial Advisor
An end-to-end Generative AI application built from scratch in Python. 
This tool leverages Large Language Models (LLMs) and real-time market data to provide automated portfolio analysis, 
technical stock insights, 
and a conversational financial interface.

🚀 Key Features
Portfolio Analyzer: Calculates annualized returns and risk (volatility) using a covariance matrix based on 5 years of 
historical data. Includes an interactive asset allocation donut chart.

Stock Technical Analysis: Fetches real-time data via yFinance to generate professional Candlestick charts and AI-driven 
technical outlooks (support/resistance, 52-week range positioning).

Conversational AI Chatbot: A stateful chat interface that maintains memory of previous interactions to provide context-aware 
financial guidance.

Professional UI: A clean, dark-themed dashboard built entirely in Streamlit with interactive Plotly visualizations.

🛠️ Tech Stack
Core Logic: Python (Pandas, NumPy)

LLM Orchestration: LangChain (LCEL, PromptTemplates)

Models: LLaMA 3.3 70B via Groq Cloud

Financial Data: yFinance

Frontend & Deployment: Streamlit

Visualizations: Plotly

📂 Project Structure

🔧 Installation & Setup

1.Clone the repository:

Bash
git clone https://github.com/YOUR_USERNAME/ai-financial-advisor.git
cd ai-financial-advisor

2.Install dependencies:

Bash
pip install -r requirements.txt
Set up your API Key:

3.Create a .streamlit/secrets.toml file and add your Groq API Key:

Ini, TOML
GROQ_API_KEY = "your_gsk_key_here"

4.Run the App:

Bash
streamlit run app.py
